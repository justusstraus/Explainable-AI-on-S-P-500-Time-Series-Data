# How to upload this to GitHub

1. Create a new repository on GitHub, for example `sp500-volatility-xai`.

2. Upload the contents of this folder:
   - `notebooks/`
   - `README.md`
   - `requirements.txt`
   - `.gitignore`

3. Do not upload the official C-SHAP source code unless you have checked and comply with its license.

4. Keep the link to the official C-SHAP repo in your README:
   https://github.com/SaxionAMI/CSHAP-for-time-series

5. After uploading, your repository should look like:

```text
sp500-volatility-xai/
├── notebooks/
│   └── sp500_volatility_xai.ipynb
├── README.md
├── requirements.txt
└── .gitignore
```
